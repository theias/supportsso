Support Single Sign On (SSO)

Gartner
Info regarding Gartner SSO.

Gartner Supports SAML 2.0 for Single Sign On (SSO).

It appears there is no information about this service provided online though support does have a confidential PDF on how to set up SAML 2.0. This document is available via support.

How to Setup SSO for Gartner

  1. Contact Gartner Support for the documentation to set up SAML
  2. The SAML configuration is similar to other configurations and the document provides the Shibboleth URN's that are required.
